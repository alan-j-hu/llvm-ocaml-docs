Module Llvm_Sparc

Sparc Initialization.

This interface provides an OCaml API for initialization of the Sparc LLVM target. By referencing this module, you will cause OCaml to load or link in the LLVM libraries corresponding to the target. By calling initialize, you will register components of this target in the target registry, which is necessary in order to emit assembly, object files, and so on.
